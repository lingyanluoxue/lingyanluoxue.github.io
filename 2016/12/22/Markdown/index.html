Markdown

Markdown 入门

Markdown 标题

标题

1.通过在行首插入 1 到 6 个 # ，来定义对应的 1 到 6 阶 标题。
2.利用 = (等号)和 -(减号)可以定义一级标题和二级标题，(任何数量的 = 和 - 都有效果) 。

Markdown 段落和换行

第一行：（结尾插入两个以上的空格后再回车）
第二行：（上一行结尾存在两个以上空格，段内换行。）相邻第一行，不存在空行，被视为同一段落。

第三行：两行之间存在空行，视为不同段落

首行缩进：
在开头的时候，先输入下面的代码，然后紧跟着输入文本即可。分号也不要掉

  • 半方大的空白&ensp;或&#8194;
  • 全方大的空白&emsp;或&#8195;
  • 不断行的空白格&nbsp;或&#160;
